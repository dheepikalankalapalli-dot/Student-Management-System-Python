# Student-Management-System-Python
A console-based Python application used to store and manage student records. The system allows users to add, view, search, update, and delete student details with persistent data storage using JSON.
# Student Management System using Python

## Project Description
This project is a simple Student Management System developed using Python. 
It allows users to manage student records through a console-based application.

## Features
- Add student details
- View all students
- Search student by roll number
- Update student details
- Delete student records

## Technologies Used
- Python
- JSON file storage

## How to Run the Project
1. Install Python
2. Download the project file
3. Run the program using:

python student_management.py
